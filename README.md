# TV.EveryWhere - APK Build Project

## Build your APK in 2 minutes (FREE)

1. **Create a GitHub repository**
   - Go to https://github.com/new
   - Name it anything, set to Public
   - Click **Create repository**

2. **Upload these files**
   - Click **"uploading an existing file"**
   - Drag ALL files from this ZIP into the upload area
   - Click **Commit changes**

3. **Get your APK**
   - Go to the **Actions** tab
   - Wait ~3 minutes for the green checkmark
   - Click the completed run → Scroll to **Artifacts**
   - Download your APK!

## Install on Android
- Transfer the APK to your phone
- Tap it → Allow "Install from unknown sources"
- Done!
